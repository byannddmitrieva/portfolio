ANNA DMITRIEVA — PORTFOLIO V2

READY FOR HOSTINGER
===================

Upload these items directly into public_html:
- index.html
- styles.css
- script.js
- assets folder

IMPORTANT
=========
Do not upload the outer anna_portfolio_site_v2 folder itself.
Inside public_html, index.html must be visible immediately.

Before launch:
1. Confirm that the Telegram, LinkedIn and email links are current.
2. Confirm that every project image can be published publicly.
3. Replace the inline favicon with platform-specific files if desired.
4. The canonical domain is set to https://a-dmitrieva.design/

Mobile:
The site includes a dedicated mobile menu, one-column case layouts,
mobile typography adjustments and touch-device fallbacks.

Local preview:
Double-click index.html, or run:
python3 -m http.server 8000
