ANNA DMITRIEVA — PORTFOLIO V2

READY FOR HOSTINGER
===================

Upload these items directly into public_html:
- index.html
- styles.css
- script.js
- assets folder

IMPORTANT
=========
Do not upload the outer anna_portfolio_site_v2 folder itself.
Inside public_html, index.html must be visible immediately.

Before launch:
1. In index.html, replace the placeholder LinkedIn and Telegram href="#".
2. Confirm the email address.
3. Confirm that every project image can be published publicly.
4. Add favicon files if desired.
5. The canonical domain is set to https://a-dmitrieva.design/

Mobile:
The site includes a dedicated mobile menu, one-column case layouts,
mobile typography adjustments and touch-device fallbacks.

Local preview:
Double-click index.html, or run:
python3 -m http.server 8000
